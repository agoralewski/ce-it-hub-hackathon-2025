# Warehouse Application Documentation

Welcome to the documentation for the Warehouse Application! This guide will help you understand how to use the main features of the application.

## Main Features
- User authentication and registration
- Inventory management (add, move, remove items)
- QR code generation and scanning
- Exporting inventory data
- User profile management
- Admin features for managing categories, racks, rooms, and shelves

Use the navigation on the left to explore detailed guides for each feature.
