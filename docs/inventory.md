# Inventory Management

- **Add Item**: Use the 'Add Item' button to add new inventory items.
- **Move Items**: Move items between locations (rooms, racks, shelves) using the 'Move Items' feature.
- **Remove Item**: Remove items from inventory when needed.
- **Low Stock Alerts**: View items that are low in stock from the dashboard or 'Low Stock' page.
- **Categories**: Organize items by categories for easier management.
